# 🎓 Student Portal Web App

A complete **Student Portal System** built using frontend and backend web technologies.
This project allows students, teachers, and admins to manage and access academic information efficiently.

---

## 🚀 Features

### 👨‍🎓 Student

- Login system
- View dashboard
- Access personal academic details

### 👨‍🏫 Teacher

- Login system
- Manage student data
- View assigned information

### 🛠️ Admin

- Admin login
- Full dashboard access
- Manage users (students & teachers)

---

## 🛠️ Tech Stack

- **Frontend:** HTML, CSS, JavaScript
- **Backend:** JavaScript (Node.js - if applicable)
